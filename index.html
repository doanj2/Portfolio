About Me

As a quick introduction, I am a second-year student at McMaster University pursuing a Bachelor of Engineering specializing in Software engineering. Although I am in my second year, I have been in the field of technology and have been creating innovative projects for the past five years. This includes developing programs for school assignments, solid modeling, circuitry, and website development. From these experiences, I have been able to learn and improve many skills that I believe are major assets required of a Software Developer.

Projects

Reycyling System

Fall 2020

This project had student-led teams design and code a recycling system that could sort between different materials and properly dispose of them. Here, our design solution was in the top 10% of all teams chosen for creating a new recycling system.

Surgical Sterilization

Fall 2020

Created a robotic sterilization system for medical tools allowing for sanitized remote surgeries. This project utilized Python to control the Q-Arm and interact with the environment. Autodesk Inventor was also used to 3D model containers for easy sanitization.
